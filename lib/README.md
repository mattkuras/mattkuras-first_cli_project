
# CLI Project

This repo provides a CLI to view the statistics re Covid-19. All information is via worldometer.com.

## Installation

clone the repo 
install gem pry
install gem nokogiri
install gem open-uri


## Usage

Type the below and follow the on screen prompts.

    /bin/executable
